libYafaRay-Gui uses CMake for the building process in Linux, Windows and MacOSX systems.

Dependencies:
  * Required
    * C++11 compliant compiler
    * \>=cmake-3.18
    * \>=libYafaRay-4.0

  * Optional, depending on options selected in CMake:
    * \>=libYafaRay-Xml-4.0
    * \>=qt-5
    * \>=wxWidgets-3.0
